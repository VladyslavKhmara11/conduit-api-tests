Conduit – API Test Automation

Automated testing of the Conduit REST API, covering user authentication, article management, comments, and user interactions.

Features

Automated API tests for user registration and login
Creating, deleting, and retrieving articles
Adding and deleting comments
Following and unfollowing users
Negative tests, including:
Invalid password or email during registration
Attempt to delete an article by a user without access rights
Validation of response status codes and data integrity

Getting Started

Clone the repository: git clone https://github.com/VladyslavKhmara11/conduit-api-tests
Open the project in Postman
Run the collection to execute automated API tests
Review test results for both positive and negative scenarios
